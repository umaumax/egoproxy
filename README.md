# egoproxy
easy go proxy

## How to install

```
go get -u github.com/umaumax/egoproxy
```

## How to run
```
egoproxy
```

## help
```
Usage of egoproxy:
  -p string
    	HTTP proxy service address (e.g., ":1080") (default ":1080")
  -verbose
    	should every proxy request be logged to stdout (default true)
```
